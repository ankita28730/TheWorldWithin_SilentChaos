# The World Within: Silent Chaos

## Overview

The World Within: Silent Chaos is a symbolic 2D psychological exploration game developed in Unity.

The game represents anxiety through an abandoned city whose atmosphere reflects the protagonist's state of mind. Players progress by making mindful choices and completing an interactive grounding journey inspired by the 5-4-3-2-1 grounding technique.

The goal is not to defeat enemies, but to regain control by reclaiming what anxiety has taken away.

---

## Features

- Multi-scene exploration
- Symbolic storytelling
- 5-4-3-2-1 grounding gameplay
- Persistent progression between scenes
- Dynamic objective system
- Choice-driven introduction
- Atmospheric visual transitions
- Context-aware interactions
